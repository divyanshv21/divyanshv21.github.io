Course Description

This course covers the concepts and techniques of non-linear regression models. Topics include: understanding the difference between linear and non-linear regression, fitting non-linear models, parameter estimation, assessing model adequacy, hypothesis testing, and model selection. Applications and examples from various fields will be discussed.
